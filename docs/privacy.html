Discogs Bulk Listener — Privacy Policy

Discogs Bulk Listener is a free, open-source Chrome extension built for the record collecting community.

Data we collect

None. The extension stores only what you enter — your YouTube OAuth Client ID, your Discogs Personal Access Token, and a YouTube access token obtained during sign-in. All of this is stored locally in your browser using Chrome's storage.local API and never transmitted to any server controlled by this extension.

Third-party services

The extension communicates directly with the YouTube Data API and Discogs API on your behalf, using credentials you provide. Their respective privacy policies apply.

No analytics, no tracking, no ads.

Questions? Open an issue on GitHub.
