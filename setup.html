Discogs Bulk Listener First-Time Setup Guide
1. Google Cloud
2. Discogs Token
3. Connect YouTube

This is a one-time setup — about 10 minutes. Once done you'll never need to do it again.

You need two things: a YouTube OAuth Client ID (lets the extension create playlists on your account) and a Discogs Personal Access Token (lets the extension access your cart and wantlist). Both are free.

Heads up: If this feels complicated and uncomfortable — sorry about that. This is a DIY project and a free extension, so unfortunately the setup has to be a bit clunky. There's nothing weird going on, it's just annoying. Follow the steps below and you'll be done in about 10 minutes.
Part 1
Google Cloud Setup
1

Create a Google Cloud project

Go to console.cloud.google.com and sign in with your Google account.

Click the project name in the top bar, then click New Project in the top-right corner of the dialog that appears.

Name it Discogs Bulk Listener (or anything you like) and click Create.

Once created you'll be taken to the project dashboard. Confirm the project name shows at the top.

2

Enable the YouTube Data API v3

Click APIs & Services from the Quick access section on the dashboard. You'll see this sidebar:

Click Library in the left sidebar.

Search for YouTube Data API v3.

Click the YouTube Data API v3 result, then click Enable.

You'll see the API dashboard confirming it's enabled.

3

Configure the OAuth consent screen

Click OAuth consent screen in the left sidebar. You'll see the Google Auth Platform overview.

Click Get started.

App Information: Enter any app name (e.g. "Discogs Bulk Listener") and select your email as the support contact. Click Next.

Audience: Select External, then click Next.

Contact Information: Enter your email address, then click Next.

Finish: Check the I agree to the Google API Services: User Data Policy box, click Continue, then Create.

You'll be returned to the OAuth Overview, now showing a Create OAuth client button.

4

Add yourself as a test user

Click Audience in the left sidebar. Scroll down to the Test users section and click + Add users.

Enter your Google account email address and click Save.

Your email will appear in the list.

5

Create an OAuth Client ID

Click Clients in the left sidebar, then click + Create client.

Set Application type to Web application.

Click Create. A dialog shows your new client. Copy the Client ID — the long value ending in .apps.googleusercontent.com. You'll paste it below.

Part 2
Discogs Token
6

Get your Discogs personal access token

Go to discogs.com/settings/developers. Click Generate token.

Your token appears below the button. Copy it.

Part 3
Connect YouTube
7

Sign in with Google

Click the button below. A Google sign-in window will open — sign in with the same Google account you used in Google Cloud Console.

Expected warning: Google may show an "App isn't verified" screen. This is normal for personal credentials. Click Advanced, then Go to [app name] (unsafe) to continue.
✓

Setup Complete

Everything is configured. Close this tab, navigate to a Discogs seller, label, or artist page, and click the extension icon to get started.
